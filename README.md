Игра Sokoban

Сюжет игры - работяге Бобу стало скучно на работе, он решил поразминаться пока приедут грузовики забрать товар.

Цель игры - переместить все ящики на карте в подходящее для них место. Сложность заключается в архитектуре карты, при перемещении ящиков они могут застрять. Живых врагов на карте к вашему счастью не будет, но в игре присутствуют ловушки.

Жанр игры - головоломка

Управление:
	Игрок полностью управляется стрелочками. 
	Leftarrow  - влево, 
	Downarrow - вниз, 
	Rightarrow - вправо,
	Uparrow - вверх
