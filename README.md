#Игра Sokoban

###Сюжет игры 
Работяге Бобу стало скучно на работе, он решил поразминаться пока приедут грузовики забрать товар.

###Цель игры 
Переместить все ящики на карте в подходящее для них место. Сложность заключается в архитектуре карты, при перемещении ящиков они могут застрять. Живых врагов на карте к вашему счастью не будет, но в игре присутствуют ловушки.

###Жанр игры 
Головоломка

###Управление:
*	Игрок полностью управляется стрелочками. 
*	Leftarrow  - влево, 
*	Downarrow - вниз, 
*	Rightarrow - вправо,
*	Uparrow - вверх 

###Сущности на карте:
1.	# - стена, никаких взаимодействий
2.	? - яма, при взаимодействии моментальный проигрыш
3.	o - ящик, главная задача игры
4.	p - игрок
5.	+ - место, куда необходимо поставить ящики
6.	O - ящик на правильном месте
7.	P - игрок на месте для ящиков
